# meal-planning-dss
Personalized Nutrition &amp; Meal Planning Decision Support System using Python &amp; Streamlit


## 📌 Overview
This project is a Decision Support System (DSS) that generates personalized meal plans based on user health data, fitness goals, and dietary preferences.

---

## 🚀 Features
- BMI calculation and classification  
- Daily calorie requirement estimation  
- Personalized meal plan generation  
- Diet filtering (veg / non-veg)  
- Health condition filtering (diabetes, hypertension)  
- Nutritional breakdown (calories, protein, carbs, fat)  
- Streamlit-based interactive interface  

---

## 🛠️ Tech Stack
Python | Streamlit | Pandas | Matplotlib
